About Pima Diabetes Prediction System

This system uses machine learning to predict diabetes risk based on the famous Pima Indians Diabetes dataset. It employs a Random Forest classifier trained on historical medical data to provide accurate predictions.

Dataset Information

The Pima Indian Diabetes Dataset originally from the National Institute of Diabetes and Digestive and Kidney Diseases. The objective is to predict based on diagnostic measurements whether a patient has diabetes.

  • Source: National Institute of Diabetes and Digestive and Kidney Diseases
  • Population: Pima Indian females, at least 21 years old
  • Features: 8 medical diagnostic measurements
  • Target: Binary classification (diabetes/no diabetes)

Machine Learning Model

Algorithm

Random Forest Classifier

Model Parameters
  • n_estimators: 100
  • random_state: 42
  • Train/Test split: 80/20
Why Random Forest?
  • Handles non-linear relationships
  • Resistant to overfitting
  • Provides feature importance
  • Works well with medical data

Feature Descriptions

Feature Description Medical Significance
Pregnancies Number of times pregnant Gestational diabetes can increase future diabetes risk
Glucose Plasma glucose concentration (mg/dL) Primary indicator of blood sugar levels
Blood Pressure Diastolic blood pressure (mm Hg) High BP often associated with diabetes
Skin Thickness Triceps skin fold thickness (mm) Indicator of body fat distribution
Insulin 2-Hour serum insulin (mu U/ml) Measures insulin response to glucose
BMI Body mass index (kg/mÂ²) Obesity is a major diabetes risk factor
Diabetes Pedigree Function Genetic diabetes risk score Family history of diabetes
Age Age in years Diabetes risk increases with age

API Documentation

Available Endpoints
POST /predict - Single Prediction

Description: Predict diabetes for a single patient

Content-Type: application/json

Request Body:

{
  "Pregnancies": 6,
  "Glucose": 148,
  "BloodPressure": 72,
  "SkinThickness": 35,
  "Insulin": 0,
  "BMI": 33.6,
  "DiabetesPedigreeFunction": 0.627,
  "Age": 50
}

Response:

{
  "Pregnancies": 6,
  "Glucose": 148,
  "BloodPressure": 72,
  "SkinThickness": 35,
  "Insulin": 0,
  "BMI": 33.6,
  "DiabetesPedigreeFunction": 0.627,
  "Age": 50,
  "Prediction": 1
}
POST /batch-predict - Batch Prediction

Description: Predict diabetes for multiple patients via CSV upload

Content-Type: multipart/form-data

Form Data: file (CSV file with required columns)

Response: JSON array with predictions for each row
